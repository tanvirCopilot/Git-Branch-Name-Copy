## [1.0.0] - 2025-12-05

### Added
- Copy git branch name with a single click
- Status bar indicator when in a git repository
- Command palette integration
- Automatic updates when switching branches
- Support for any git repository

### Features
- Click on status bar icon to copy current branch name
- Visual feedback when branch name is copied
- Tooltip showing current branch name
- Automatic detection of git repositories
